# AntiLiso

Web app de chat com o agente financeiro AntiLiso.

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra a URL local no navegador. No iPhone: Safari → Compartilhar → Adicionar à Tela de Início.

## Variáveis

`VITE_API_BASE_URL` aponta para a API (padrão: `https://antiliso-api.apichatzon.online/api/v1`).
